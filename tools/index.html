External Tools

The Taskwarrior distribution contains the essential core components. The open source community provides many non-core components, distributed separately, because they are not tied to Taskwarrior release schedule.

Do you have a taskwarrior extension you'd like to see listed here? Please tell us about it.

Author: by Bryce W. Harrington. 		Unknown 2.1.2 or earlier
tchart A Gantt chart style front end Taskwarrior. Tchart allows for rapid selection and modification of tasks, while the GANTT type report enables a rapid visual overview of tasks based on their due dates.
