The Taskwarrior distribution contains the essential core components. The open source community provides many non-core components, distributed separately, because they are not tied to Taskwarrior release schedule.

Do you have a taskwarrior extension you'd like to see listed here? Do you have any corrections to suggest? Please tell us about it.
