Helper Commands

Helper commands all begin with an underscore (i.e. _ids), and they exist to provide support for third party add-ons, such as shell completion scripts.

Helper commands are not intended for regular use, but there is no reason for that to stop you.

  • _aliases
  • _columns
  • _commands
