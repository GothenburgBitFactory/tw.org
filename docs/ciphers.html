Cipher Selection

By default, Taskserver uses the GnuTLS cipher set: 

PERFORMANCE:%SERVER_PRECEDENCE
Taskwarrior uses: 
NORMAL
You may wish to override these ciphers with a set that satisfies your security needs. A vigilant Taskserver administrator may wish to respond to news of various cipher weaknesses.

A more secure set would be to use only TLS 1.2 and these ciphers: 

ECDHE-ECDSA-AES256-GCM-SHA384
If AES is compromised, perhaps use: 
DHE-RSA-CAMELLIA256-SHA
If elliptical curves are compromised, perhaps use: 
DHE-RSA-AES256-SHA

Configuration

Taskserver supports the ciphers configuration setting and Taskwarrior supports taskd.ciphers as a means to override the default set.
